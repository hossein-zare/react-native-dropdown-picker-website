Modes

Modes display selected items.

  • DEFAULT
  • SIMPLE
  • BADGE
mode="SIMPLE"
TypeDefault
stringSIMPLE
note

Single item pickers won't be affected by the mode property.

Change the default modeâ€‹

You can change the default mode of the package globally.

DropDownPicker.setMode("BADGE");
